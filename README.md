# Akari Store
Akari Store es una app web que sirve para que el cliente pueda acceder al catálogo de productos de animé de Akari. 
En la app puede acceder a las diferentes categorías de productos como cuadros, remeras, peluches o figuras de acción. 
En la tarjeta de presentación de cada producto tiene un enlace a los detalles del mismo como su precio, sus características o su stock. Desde allí puede agregar artículos a su carrito y luego seguir buscando artículos o ir al carrito.
También puede acceder al carrito desde la barra de navegación donde se detalla el número de ítems guardados.
Luego, al acceder a su carrito puede eliminar artículos, ver el total a pagar, vaciar el carrito o si está de acuerdo con su contenido puede dirigirse a generar la orden de compra.
Una vez que ingresa, es dirigido a un formulario donde debe ingresar sus datos.
Cuando envía el formulario se genera la orden de compra, se le muestra el código y el detalle de lo que compró.
Esta web app creada con React + Vite fue realizada como proyecto para el curso de React de CoderHouse.

### Guía de instalación

1. Clonar el repositorio [Link Repo](https://github.com/GEZ-31/CreaTuLandingZamboni)
2. Moverse a la carpeta creada `cd akari_store`
3. Instala las dependencias del proyecto `npm install`
4. Levantar la app en un entorno local `npm run dev`

>Es necesario tener instalada la versión de Node v22.11.0

- [React Icons](https://react-icons.github.io/react-icons/) utilizada para los íconos

- [Firebase](https://firebase.google.com/) utilizada como base de datos
